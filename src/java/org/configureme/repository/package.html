Internal configuration repository. The ConfigurationRepository is the internal presentation of attributes and their values. Each configuration is stored as an Artefact which contains Attributes, which in return contain AttributeValues. It also contains a ConfigurationImpl which is a snapshot of an Artefact in one of the Environments.
