Main package, which contains most classes a user of configureMe must know. ConfigurationManager, ConfigurationRepository and other class which are declared as enums with only one element called INSTANCE are singletons.

Package Specification

Contains main user interface classes: ConfigurationManager, Configuration, Environment.
