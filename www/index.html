Mission statement

Goals:

Create a solution for object configuration with following features:

  • Automatic type safe configuration
  • Support for cascading environments
  • One on one relation between a configuration and a configurable object
  • Low load time, low memory footprint
  • Overview of all configuration options for a given configurable object at a glance

Additional features:

  • In production site edit of config file possible
  • Automatic reconfiguration upon config file edit

If you are interested in knowing more please contact the owner at owner@configureme.dev.java.net

Check out the Project WiKi for more info
